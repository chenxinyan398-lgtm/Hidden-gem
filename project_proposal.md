# 專案企劃書：SecretSpace (MVP 核心版)

## 一、 專案概述
SecretSpace 是一個高度隱密的個人資訊整合與社交分享平台。有別於傳統社群軟體的全面公開，本軟體預設所有內容皆為「上鎖」狀態，將資訊的主導權完全還給使用者。透過實體的 QR Code 掃描或特定的數位授權，讓分享私房清單或心情日記成為一種建立信任與親密感的儀式。

## 二、 核心目標與痛點解決
- **解決痛點**：社交場合缺乏有深度的私密展示空間，現有社群過於公開，難以建立「專屬特定人士」的信任感。
- **核心目標**：打造現代版的「數位交換日記」與「深度私密名片」。

## 三、 核心功能 (MVP 階段)
為求快速驗證與開發，MVP 階段將專注於最核心的「私密記錄」與「授權解鎖」流程：

1. **私密內容管理 (預設上鎖)**
   - **私房清單 (Lists)**：記錄個人私房餐廳、景點或收藏。
   - **心情日記 (Diaries)**：撰寫個人的私密日記與筆記。
2. **授權解鎖機制 (QR Code / 授權碼)**
   - **生成 QR Code**：使用者可為特定清單或日記生成專屬解鎖 QR Code。
   - **掃描解鎖**：其他人透過掃描該 QR Code，即可獲得該內容的閱讀權限。
3. **個人私密主頁**
   - 訪客視角：只能看到對方已授權解鎖的內容列表。
   - 主人視角：可以管理所有內容（新增/編輯/刪除），並查看哪些內容已授權給誰。

## 四、 未來展望 (MVP 後階段)
- 權限分級制 (不同好友看不同等級內容)
- 限時授權與閱後即焚
- 雙向解鎖 (真正的交換日記)
- 訪客足跡與私密留言功能
